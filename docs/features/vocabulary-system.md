# Vocabulary System Rules

## Input
- Users can answer using kana or romaji.

## Accepted Romaji Formats
- KAERU
- KA-E-RU
- KA E RU

## Validation
- Validation should ignore:
  - spaces
  - hyphens
  - uppercase/lowercase differences

## Learning Flow
- Mistakes must repeat at the end.
- Final result screen must show:
  - total score
  - failed words
  - corrected answers

## Vocabulary Structure
Each entry must contain:
- id
- kana
- romaji
- meaning
- image
- mainKana
- series
